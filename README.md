# VaryShop Plugins Marketplace

A Claude plugin marketplace by Michal Varys.

## Installation

Add this marketplace to Claude via git:

```
github:michalvarys/claude-plugins
```

## Available Plugins

### Odoo Dev Toolkit
Complete toolkit for Odoo 18 development — QWeb website pages, module scaffolding, XML views, Python models, controllers, and wizards.

### Social Media Creator
Create branded social media content — static posts, animated videos, carousels, voiceovers, background music — with scheduling and automated publishing pipelines.
