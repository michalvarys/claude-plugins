Create a carousel post about: $ARGUMENTS

Follow the create-carousel skill instructions:
1. Read the skill for carousel structure guidelines
2. Plan the carousel: cover slide, 4-8 content slides, CTA slide
3. Design each slide as a separate 1080x1080px HTML file with consistent styling
4. NEVER use emoji characters — always use inline SVG icons
5. Maintain visual consistency across all slides (same colors, fonts, branding)
6. Render all slides to PNG using Playwright
7. Generate post text for all platforms
8. Save all HTML and PNG files to the outputs folder
9. Show the user previews of all slides
